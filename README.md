# Futnet

Futnet is a Flutter-based mobile application that allows users to explore football teams, view player profiles, and keep track of team details. This project serves as a foundation for a football management or fan engagement app.

## Features

- Browse a list of football teams.
- View detailed information about each team.
- Explore player profiles with stats and positions.
- Clean and responsive Flutter UI.
- Fully compatible with both Android and iOS platforms.
